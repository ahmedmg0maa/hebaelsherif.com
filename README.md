# Heba El Sharif Premium Website

نسخة احترافية كاملة لموقع **هبة الشريف**، مبنية بـ Next.js و Tailwind CSS، عربية أولًا مع واجهة إنجليزية مبدئية، Dark Mode، متجر/Checkout، نماذج API، وصفحات SEO وقانونيات جاهزة.

## التشغيل

```bash
npm install
npm run dev
```

ثم افتح:

```txt
http://localhost:3000
```

## أهم ما تم تطويره

- هوية بصرية كاملة بألوان البراند: الكريمي، الرملي، التركواز، الذهبي، الزيتوني، والعنابي.
- واجهة عربية RTL مع صفحة إنجليزية مبدئية `/en`.
- Dark Mode باستخدام `next-themes`.
- Header احترافي: اللوجو يمين، القائمة في المنتصف، أزرار التحكم والحجز شمال.
- صفحات كاملة: الرئيسية، من هي هبة، الخدمات، الكورسات، تفاصيل الكورس، الكتب، المقالات، تفاصيل المقال، الحجز، التواصل، تجارب العملاء، الإعلام، الحساب.
- صفحة Checkout مستقلة للكتب والكورسات: `/checkout`.
- API Routes تجريبية جاهزة للربط الحقيقي:
  - `POST /api/contact`
  - `POST /api/booking`
  - `POST /api/checkout`
- صفحات قانونية جاهزة للتخصيص:
  - `/privacy`
  - `/terms`
- SEO جاهز:
  - `app/sitemap.ts`
  - `app/robots.ts`
  - Metadata و OpenGraph في `app/layout.tsx`
- صور وهوية البراند داخل `public/images`.

## متغيرات البيئة

انسخ الملف:

```bash
cp .env.example .env.local
```

ثم أضف مفاتيح الخدمات المطلوبة عند الربط الحقيقي.

## الربط الحقيقي لاحقًا

- **Paymob** للدفع داخل مصر.
- **Stripe** للدفع الدولي.
- **Supabase** للحسابات، الطلبات، الكورسات، ولوحة الإدارة.
- **Resend** لإرسال رسائل تأكيد الحجز والشراء.

النسخة الحالية تعمل Demo Mode بدون مفاتيح خارجية، بحيث تقدر تختبر تجربة المستخدم بالكامل قبل الربط.

## المرحلة 3 — AI + Account + Performance

تمت إضافة تحديثات جديدة:

- مساعد ذكاء اصطناعي عائم في كل صفحات الموقع.
- API Route للمساعد: `POST /api/assistant`.
- يعمل Demo Mode بدون مفاتيح، وعند إضافة `OPENAI_API_KEY` يستخدم OpenAI Responses API.
- صفحة الحساب أصبحت Dashboard كاملة تشمل: الكورسات، الجلسات، الطلبات، الفواتير، التحميلات، البروفايل، وتفضيلات التواصل.
- تحسينات سرعة: ضغط الصور، تحويل اللوجو إلى WebP، تحسين Next Image، تعطيل `x-powered-by`، ضغط الاستجابات، وتحسين تحميل أقسام الصفحات.
- إضافات تفاعلية: Chat quick prompts، حفظ محادثة المساعد محليًا، Progress للكورسات، ملاحظات يومية، وتفضيلات حساب قابلة للتغيير.

### تفعيل المساعد الذكي الحقيقي

انسخ `.env.example` إلى `.env.local` وأضف:

```bash
OPENAI_API_KEY=your_key_here
OPENAI_MODEL=gpt-4.1-mini
```

بدون المفتاح سيعمل المساعد بردود ذكية تجريبية مبنية على محتوى الموقع.
